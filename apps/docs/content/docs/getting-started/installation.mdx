## Prerequisites

- Bun installed on your machine.
- PostgreSQL available locally or remotely.

## Install packages

```bash
bun add bunbase
```

For React client projects:

```bash
bun add @bunbase/react @tanstack/react-query
```

## Bootstrap a new app

```bash
bunx bunbase init my-app
cd my-app
bun install
```
