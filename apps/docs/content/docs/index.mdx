## What Bunbase gives you

- Type-safe action execution with schema validation.
- Trigger-based invocation via API, cron, events, and webhooks.
- Built-in guards for authentication and authorization.
- Integrated OpenAPI generation and typed React client support.
- A Bun-first runtime and CLI workflow.
