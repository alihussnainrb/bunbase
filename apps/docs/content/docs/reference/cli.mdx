---
title: CLI Commands
description: Bunbase CLI command reference.
---

## Development

```bash
bunbase dev
bunbase generate action <name>
bunbase generate module <name>
```

## Database

```bash
bunbase migrate
bunbase migrate new <name>
bunbase migrate status
bunbase typegen:db
```

## React type generation

```bash
bunbase typegen:react --url http://localhost:3000
```

## Project setup

```bash
bunbase init <name>
```
